Question-1: What are the different ways to select an element in the DOM?

There are some ways to select a HTML Element in DOM in JavaScript. These are : getElementById(), getElementsByClassName(), getElementsByTagName(), querySelector(), querySelectorAll(), getElementsByName().

Question-2: What is the difference between innerHTML, innerText, and textContent ?

The innerHTML returns or set the whole content of element including HTML tag. The innerText only returns the text of element and ignores HTML tag. The textContent returns all the text of element and ignores HTML tag.

Question-3: What is event delegation in the DOM?

Event Delegation is a technique where event listener is set on a parent so that the event also works on its child.

Question-4: What is event bubbling in the DOM?

Event bubbling is a process where if a event is triggered in a child element that event first works on the child element then gradually on its parents and then on the upward all parents.

Question-5: How do you create, add, and remove elements using JavaScript?

In JavaScript to create an element we use document.createElement(), to add an element parent.appendChild(), to remove an element parent.removeChild().
